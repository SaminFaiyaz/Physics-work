# Physics Simulation & Computation Suite

A collection of interactive web-based calculators and computational physics scripts designed to solve, analyze, and visualize core physical concepts.

---

## 🚀 Interactive Web Simulations (HTML/CSS/JS)
These tools are built using native web technologies and styled according to standardized academic frameworks.

### 1. Optical Lens & Mirror Calculator
* **Location:** `interactive-simulations/optics-calculator/index.html`
* **Features:** * Solves for $u$, $v$, or $f$ dynamically using the standard Gaussian formula.
  * Adheres strictly to standard textbook conventions (automatic negative focal lengths for diverging elements and modulus magnification $|m|$).
  * Computes lens power ($P$) in Diopters.
